# Vyra E-Commerce Website

---
# Project Overview
Vyra is a front-end e-commerce website built using HTML, CSS, and JavaScript. 
The project focuses on creating a visually appealing interface and user-friendly navigation to simulate a real online store.

---

# Live Demo Link
- GitHub Pages:  https://anothile-b.github.io/Vyra-Ecommerce-App/ 

---

# Features
- Attractive homepage with featured products
- Product listing and browsing
- Product details view
- Add to cart functionality
- Easy navigation across pages
- Responsive design for mobile and desktop
- Clean and modern UI
- Checkout and payment integration

---

# Technologies Used
- HTML5 – Structure and layout
- CSS3 – Styling and responsiveness
- JavaScript – Interactivity and functionality

---

# Purpose
This project was created to:
- Practice front-end web development
- Build a real-world e-commerce interface
- Improve JavaScript functionality skills
- Enhance UI/UX design techniques
